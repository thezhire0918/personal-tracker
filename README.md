# Personal Tracker

## Description
A simple app to track income, expenses, and remaining balance (willl add more).

## Features
- User registration and login
- Add income and expenses
- View transaction history
- Compute remaining balance

## Tech Stack
- Python (FastAPI)
- SQLite
- JWT Authentication

## How to Run
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
